# 四大品类输出格式示例

本文档展示四大品类（儿童安全座椅、婴儿推车、儿童高脚椅、儿童床）的专业输出格式，所有输出均符合工程师工作流，可直接用于CAD建模、测试方案制定、供应商对接。

---

## 1. 儿童安全座椅输出示例

### ECE R129标准

```
📦 儿童安全座椅设计方案（严格遵守ECE R129:2021 (欧盟i-Size)）
【适用标准】ECE R129:2021 (欧盟i-Size)
标准版本：2021版 | 实施要求：欧盟强制实施
🔍 核心要求：动态碰撞三向覆盖，侧防系统强制，ISOFIX接口兼容ISO 14530-3

📊 基础适配数据（基于用户输入身高：90cm）
🔽 假人参数（ISO 13232-2:2021）
▫️ 假人模型：ECE R129 Q3假人
▫️ 百分位/年龄：50th百分位3-4岁儿童
▫️ 身高范围：87-105cm（用户输入90cm处于该范围中值，适配性最优）
▫️ 体重范围：13-18kg
▫️ 人体测量参数：坐高52cm，肩宽28cm，头围49cm
▫️ 安装方向：ECE R129: 后向 / 前向 / 增高垫

📏 设计参数（GPS028-2023数据库 + 标准强制要求）
▫️ 头枕高度：535-585mm（基准点：坐骨结节（H点），公差：±5mm）
▫️ 座宽：有效座宽：350mm，总座宽（含侧防）：420mm
▫️ ISOFIX Envelop（盒子）尺寸：ISOFIX Size Class B2（ECE R129 §5.3.2 / GB 27887-2024 §5.2）
▫️ Envelop详细尺寸：纵向长度730mm(±10mm)，横向宽度460mm(±5mm)，固定点间距300-350mm
▫️ 侧防面积要求：≥0.85m²（覆盖T12胸部至P8头部侧方区域）
▫️ 侧防测试标准：EN 14154-3:2022

⚖️ 测试要求（量化阈值 + 标准条款，可直接用于测试方案）
▫️ 正面碰撞：碰撞速度50km/h(±1km/h)，碰撞台加速度15g(持续3ms)，HIC≤1000（ECE R129 §7.1.2）
▫️ 侧撞胸部压缩：侧撞速度60km/h(移动壁障)，胸部压缩量≤44mm，压缩速度≤2.5m/s（ECE R129 §7.1.3）
▫️ 安全带织带强度：纵向≥26.7kN，横向≥17.8kN（测试方法：ISO 6683:2017）

🧪 标准测试项（测试设备+流程+合格判据，可直接对接实验室）
动态碰撞：正碰
   测试设备：HYGE电动碰撞台（符合ISO 6487:2018）
   测试条件：假人后向，约束系统：ISOFIX+Top Tether，碰撞速度50km/h
   ✅ 合格判据：头部HIC≤1000，胸部加速度≤60g（3ms滑动平均）

动态碰撞：侧碰
   测试设备：移动壁障（符合ECE R129 §7.1.3）
   测试条件：侧向撞击速度60km/h，使用Q3假人
   ✅ 合格判据：胸部压缩≤44mm，腹部受力≤2.5kN
```

## 2. 婴儿推车输出示例

### EN 1888标准

```
📦 婴儿推车设计方案（严格遵守EN 1888:2022 (欧盟强制)）
【适用标准】EN 1888:2022 (欧盟强制)
标准版本：2022版 | 实施要求：欧盟强制实施
🔍 核心要求：稳定性≥15°，双轮锁死刹车，折叠尺寸≤100×60×40cm

📊 基础适配数据（基于用户输入身高：90cm / 体重：14kg）
🔽 儿童适配参数（ISO 7176-5:2017）
▫️ 身高范围：EN 1888: 新生儿-105cm（0-4岁）
▫️ 体重范围：EN 1888: 0-15kg
▫️ 年龄范围：EN 1888: 0-48个月
▫️ 人体测量参数：坐高40cm，肩宽24cm，头围46cm
▫️ 适用场景：日常出行、公园散步、旅行

📏 设计参数（推车结构设计标准）
▫️ 轴距：轴距≥500mm（确保推车稳定性）
▫️ 刹车类型：双轮锁死制动系统，操作力≤50N
▫️ 折叠尺寸：折叠后尺寸≤100×60×40cm（便于携带）
▫️ 稳定角：稳定角≥15°（前后）/≥10°（左右）
▫️ 扶手高度：扶手高度950-1100mm（可调节）

⚖️ 测试要求（量化阈值 + 标准条款，可直接用于测试方案）
▫️ 稳定性：前后倾斜15°无倾倒，左右倾斜10°无倾倒（EN 1888 §5.7）
▫️ 刹车强度：制动力≥150N，在10°斜面上无滑动（EN 1888 §5.8）
▫️ 折叠强度：反复折叠500次无卡滞，折叠力≤50N（EN 1888 §5.9）

🧪 标准测试项（测试设备+流程+合格判据，可直接对接实验室）
稳定性测试
   测试设备：倾斜台（符合EN 1888 §5.7）
   测试方法：前后左右倾斜测试，分别倾斜15°/10°，保持30秒
   ✅ 合格判据：无倾倒，座椅无滑动，锁定机构无失效

刹车测试
   测试设备：斜坡测试台
   测试方法：在10°斜坡上施加150N制动力，保持10秒
   ✅ 合格判据：车轮无滚动，刹车机构无失效，制动距离≤50mm

折叠耐用性测试
   测试设备：折叠测试机
   测试方法：反复折叠500次，每次折叠力≤50N
   ✅ 合格判据：机构无卡滞，无零件脱落，无永久变形
```

## 3. 儿童高脚椅输出示例

### EN 14988标准

```
📦 儿童高脚椅设计方案（严格遵守EN 14988:2021 (欧盟强制)）
【适用标准】EN 14988:2021 (欧盟强制)
标准版本：2021版 | 实施要求：欧盟强制实施
🔍 核心要求：稳定性≥10°，五点式安全带，托盘载荷≥30kg

📊 基础适配数据（基于用户输入身高：90cm / 体重：14kg）
🔽 儿童适配参数（ISO 7176-9:2009）
▫️ 身高范围：EN 14988: 75-120cm（6个月-3岁）
▫️ 体重范围：EN 14988: 6-15kg
▫️ 年龄范围：EN 14988: 6-36个月
▫️ 人体测量参数：坐高38cm，肩宽22cm，头围48cm
▫️ 适用场景：家庭用餐、餐厅辅助、学习桌配套

📏 设计参数（高脚椅结构设计标准）
▫️ 座椅高度调节范围：座椅高度可调节范围750-950mm（3档调节）
▫️ 安全带类型：五点式安全带，织带宽度≥25mm
▫️ 与桌面边缘距离：与桌面边缘距离≥200mm（防夹手）
▫️ 防倾倒尺寸：座面深度≥300mm，座面宽度≥350mm

⚖️ 测试要求（量化阈值 + 标准条款，可直接用于测试方案）
▫️ 稳定性：前后倾斜10°无倾倒，左右倾斜15°无倾倒（EN 14988 §5.5）
▫️ 安全带强度：织带断裂强度≥200N，卡扣保持力≥250N（EN 14988 §5.6）
▫️ 托盘强度：托盘载荷≥30kg无断裂，边缘无锐利边（EN 14988 §5.7）

🧪 标准测试项（测试设备+流程+合格判据，可直接对接实验室）
稳定性测试
   测试设备：倾斜台（符合EN 14988 §5.5）
   测试方法：前后左右倾斜测试，分别倾斜10°/15°，保持30秒
   ✅ 合格判据：无倾倒，座椅无滑动，锁定机构无失效

安全带测试
   测试设备：拉力试验机
   测试方法：施加200N拉力于安全带，保持10秒
   ✅ 合格判据：织带无断裂，卡扣无松脱，永久变形≤5mm

托盘载荷测试
   测试设备：静载荷测试台
   测试方法：托盘上施加30kg载荷，保持60秒
   ✅ 合格判据：托盘无断裂，支撑结构无永久变形
```

## 4. 儿童床输出示例

### EN 716标准

```
📦 儿童床设计方案（严格遵守EN 716:2021 (欧盟强制)）
【适用标准】EN 716:2021 (欧盟强制)
标准版本：2021版 | 实施要求：欧盟强制实施
🔍 核心要求：防护栏高度≥600mm，防护栏间距≤60mm，床垫厚度≤100mm

📊 基础适配数据（基于用户输入身高：90cm / 体重：14kg）
🔽 儿童适配参数（ISO 7176-16:2012）
▫️ 身高范围：EN 716: 50-150cm（新生儿-5岁）
▫️ 体重范围：EN 716: 0-36kg
▫️ 年龄范围：EN 716: 0-60个月
▫️ 人体测量参数：坐高35cm，肩宽20cm，头围48cm
▫️ 适用场景：家庭卧室、婴儿房、儿童房

📏 设计参数（儿童床结构设计标准）
▫️ 防护栏高度：防护栏高度≥600mm（从床垫表面测量）
▫️ 防护栏间距：防护栏间距≤60mm（防夹头）
▫️ 床垫厚度：床垫厚度≤100mm（防窒息）
▫️ 床架强度：床架强度：静态载荷≥800N

⚖️ 测试要求（量化阈值 + 标准条款，可直接用于测试方案）
▫️ 防护栏强度：防护栏载荷≥400N，无断裂无变形（EN 716 §5.4）
▫️ 间距要求：防护栏间距≤60mm（使用φ25mm测头）（EN 716 §5.2）
▫️ 床垫适配：床垫与床架间隙≤25mm（防卡陷）（EN 716 §5.3）

🧪 标准测试项（测试设备+流程+合格判据，可直接对接实验室）
防护栏强度测试
   测试设备：拉力试验机（符合EN 716 §5.4）
   测试方法：防护栏中部施加400N垂直载荷，保持30秒
   ✅ 合格判据：防护栏无断裂，无永久变形≤2mm

防护栏间距测试
   测试设备：φ25mm测头（符合EN 716 §5.2）
   测试方法：使用φ25mm测头检测所有防护栏间距
   ✅ 合格判据：所有间距≤60mm，无卡陷风险

床垫适配测试
   测试设备：间隙测量尺
   测试方法：测量床垫与床架四周间隙，取最大值
   ✅ 合格判据：最大间隙≤25mm，无卡陷风险
```

## 输出格式统一规范

所有品类的输出格式遵循以下统一规范：

### 1. 标题部分
- 格式：`📦 [产品类型]设计方案（严格遵守[标准名称]）`
- 要求：明确产品类型和适用标准

### 2. 适用标准部分
- 格式：
  ```
  【适用标准】[标准名称]
  标准版本：[版本] | 实施要求：[实施要求]
  🔍 核心要求：[核心要求]
  ```
- 要求：包含标准名称、版本、实施要求、核心要求

### 3. 基础适配数据部分
- 格式：
  ```
  📊 基础适配数据（基于用户输入身高：[身高] / 体重：[体重]）
  🔽 [儿童/假人]适配参数（[ISO标准]）
  ▫️ [参数名称]：[参数值]
  ...
  ```
- 要求：包含身高、体重、年龄、人体测量参数、适用场景

### 4. 设计参数部分
- 格式：
  ```
  📏 设计参数（[设计标准]）
  ▫️ [参数名称]：[参数值]（[备注]）
  ...
  ```
- 要求：包含关键设计参数、量化阈值、公差要求

### 5. 测试要求部分
- 格式：
  ```
  ⚖️ 测试要求（量化阈值 + 标准条款，可直接用于测试方案）
  ▫️ [测试项目]：[测试要求]（[标准条款]）
  ...
  ```
- 要求：包含测试项目、量化阈值、标准条款引用

### 6. 标准测试项部分
- 格式：
  ```
  🧪 标准测试项（测试设备+流程+合格判据，可直接对接实验室）
  [测试名称]
     测试设备：[设备名称]（符合[标准条款]）
     测试方法：[测试流程]
     ✅ 合格判据：[合格判据]
  ...
  ```
- 要求：包含测试设备、测试方法、合格判据

## 特点总结

1. **专业性**：所有输出均符合工程师工作流，可直接用于CAD建模、测试方案制定
2. **可追溯**：所有参数均引用标准条款，便于验证
3. **量化明确**：所有阈值均为量化值，无模糊表述
4. **统一格式**：四大品类输出格式统一，便于工程师快速理解
5. **全链路覆盖**：从标准选择到测试要求，完整覆盖设计全流程
