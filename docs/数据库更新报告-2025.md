## 标准更新详情

### 1. ECE R129 (i-Size) 标准

**标准状态**：Current（2025年广泛应用）

**更新内容**：
- 标准名称更新为包含 "i-Size" 标识
- 2025年应用愈发广泛，成为衡量新生儿安全座椅质量的重要标准
- 适用年龄明确标注为根据身高分组
- 核心范围更新：相比ECE R44，新增侧面碰撞测试，采用身高分组，强制ISOFIX，反向安装延长至15个月，使用Q系列假人

**关键特性**：
- 基于身高分组（非体重）
- 强制使用ISOFIX硬连接接口
- 延长反向安装至15个月
- 新增侧面碰撞测试
- 采用新一代Q系列假人

**文件更新**：
- `EceR129StandardDatabase.kt` - 更新标准信息、修复语法错误（const val -> val，中文字段名 -> 英文字段名）

### 2. FMVSS 213/213a 标准

**标准状态**：Current（2025年重要更新）

**FMVSS 213a 更新内容**：
- **更新时间**：2025年
- **合规截止日期**：2026年12月5日（原定2025年6月30日）
- **适用范围**：体重≤40lb（约18.1kg）的儿童安全座椅
- **测试速度**：30mph（48km/h）- 接近美国常见城市限速
- **测试类型**：首次引入侧面冲击评估
- **假人类型**：CRABI 12（12个月假人）、Q3s（3岁侧撞假人）

**关键更新点**：
- 新增侧碰测试要求
- 测试速度设定为30mph
- 适用于体重≤40lb的儿童安全座椅
- 合规截止日期延长至2026年12月5日

**文件更新**：
- `FMVSSDatabase.kt` - 更新FMVSS 213a标准信息、侧面碰撞测试配置

### 3. 高脚椅标准

#### EN 14988-1:2024

**标准状态**：Current（2024年最新版）

**更新内容**：
- 最新版本：EN 14988:2024
- 发布机构：意大利统一标准化组织
- 丹麦标准化协会发布 DANSK DS/EN 14988+A2:2024 也属于该标准体系
- 适用：3岁以下能独立坐着的儿童
- 涵盖：家用和非家用（除医用特殊高脚椅外）

#### ASTM F404-21

**标准状态**：Current（2021年最新版）

**更新内容**：
- 最新版本：ASTM F404-21
- 化学要求更新：
  - 高脚椅不得使用含铅量超标的油漆喷涂
  - 可触及部件总铅含量有严格限制
- 2024年相关标准（如ASTM F404 – 21）持续发挥作用

**文件更新**：
- `HighChairStandardDatabase.kt` - 更新EN 14988和ASTM F404标准信息，修复语法错误

### 5. GPS028 / GB 28007 标准

**标准状态**：Current（2024年最新版）

**GB 28007-2024 婴幼儿及儿童家具安全技术规范**

**发布信息**：
- 发布日期：2024年6月25日
- 实施日期：2026年1月1日

**更新内容**：
- 全面整合并替代了GB 22793.1-2008、GB 28007-2011等多项旧标准
- 覆盖范围更广泛：适用于0-14岁婴幼儿及儿童使用的家具
- 明确区分：
  - "婴幼儿家具"（36个月及以下）
  - "儿童家具"（3-14岁）
  - 并设定差异化要求

**核心内容**：
- **安全要求更严格**：聚焦"防伤害+防隐患"双重目标
  - 结构安全
  - 材料安全
  - 电气安全
- **标识要求更规范**：强化全流程安全提示
  - 产品标识
  - 使用说明
  - 字体规范

**文件更新**：
- `GPS028Database.kt` - 添加GB 28007-2024标准信息，更新标准类型枚举

## 代码修复

### 语法错误修复

所有数据库文件中发现的 `const val` 语法错误已修复：

1. **EceR129StandardDatabase.kt**
   - 修复 `const val` -> `val`
   - 修复中文字段名 -> 英文字段名
     - `适用车型` -> `applicableModels`
     - `安装方向` -> `installationDirection`
     - `速度范围` -> `velocityRange`
     - `速度标准` -> `velocityStandard`
     - `刹车距离` -> `brakingDistance`
     - `适用场景` -> `applicableScenario`

2. **HighChairStandardDatabase.kt**
   - 修复 `const val` -> `val`

3. **CribStandardDatabase.kt**
   - 修复 `const val` -> `val`
