# 输出格式优化说明

## 更新概述

已将儿童安全座椅的设计方案输出格式优化为与专业案例一致的格式，使其更加专业和详细。

## 之前的输出格式（简化版）

```
📊 基础适配数据（基于用户输入身高：90cm）
🔽 假人参数（ISO 13232-2:2021）
▫️ 假人模型：Q3假人
▫️ 百分位/年龄：50%百分位3岁儿童
▫️ 身高范围：87-105cm（用户输入90cm处于该范围中值，适配性最优）
▫️ 体重范围：13-18kg
▫️ 人体测量参数：坐高52cm，肩宽28cm，头围49cm
▫️ 安装方向：后向（动态碰撞三向覆盖，侧防系统强制）

📏 设计参数（GPS028-2023数据库 + 标准强制要求）
▫️ 头枕高度：535-585mm（基准点：坐骨结节（H点），公差：±5mm）
▫️ 座宽：有效座宽：350mm，总座宽（含侧防）：420mm
...
```

## 新的输出格式（专业版）

```
📦 儿童安全座椅设计方案（产品型号：CS-2025-X）

├─ 【适用标准】ECE R129:2021 (欧盟i-Size) + GB 27887-2024（中国新标，2024版强制实施）（醒目蓝色标签）
│  🔍 标准关键要求：动态碰撞测试覆盖正/侧/后三向，侧防系统强制要求，ISOFIX接口兼容ISO 14530-3
│
├─ 📊 基础适配数据（基于选中标准+用户输入身高：90cm）
│  ├─ 🔽 假人参数（专业级假人配置）
│  │  ├─ 假人模型：ECE R129 Q3 假人（50th百分位3岁儿童，标准编号：ISO 13232-2:2021）
│  │  ├─ 身高范围：87-105cm（用户输入90cm处于该范围中值，适配性最优）
│  │  ├─ 体重范围：13-18kg（推荐设计载荷取15kg，覆盖50%百分位）
│  │  ├─ 人体测量参数：坐高52cm，肩宽28cm，头围49cm（GPS028-2023数据库校准值）
│  │  └─ 安装方向：后向（ECE R129要求≤105cm儿童优先后向，GB 27887-2024强制要求）
│
├─ 📏 设计参数（GPS028-2023数据库 + 标准强制要求）
│  ├─ 头枕高度：535-585mm（调节范围，基准点：坐骨结节（H点），公差±5mm）
│  ├─ 座宽：有效座宽350mm（臀部支撑区域），总座宽420mm（含侧防侧翼）
│  ├─ ISOFIX Envelop尺寸（盒子尺寸）：ISOFIX Size Class B2（ECE R129 §5.3.2）
│  │  - 纵向长度：730mm（±10mm），横向宽度：460mm（±5mm）
│  │  - 固定点间距：300-350mm（兼容主流车型ISOFIX接口布局）
│  └─ 侧防面积：≥0.85m²（覆盖假人胸部（T12）至头部（P8）侧方区域，EN 14154-3:2022测试要求）
│
├─ ⚖️ 测试要求（标准条款+量化阈值，可直接用于测试方案制定）
│  ├─ 正面碰撞：碰撞速度50km/h（±1km/h），碰撞台加速度15g（持续3ms），HIC≤1000（ECE R129 §7.1.2）
│  ├─ 侧撞胸部压缩：侧撞速度60km/h（移动壁障），胸部压缩量≤44mm，压缩速度≤2.5m/s（GB 27887-2024 §6.7）
│  └─ 安全带织带断裂强度：≥26.7kN（纵向），≥17.8kN（横向），测试方法：ISO 6683:2017
│
└─ 🧪 标准测试项（含测试流程+合格判据，可直接对接实验室测试）
   ├─ 动态碰撞：正碰
   │  - 测试设备：HYGE电动碰撞台（符合ISO 6487:2018）
   │  - 碰撞姿态：假人后向，约束系统：ISOFIX+Top Tether
   │  - 合格判据：假人头部HIC≤1000，胸部加速度≤60g（3ms滑动平均）
   ├─ 动态碰撞：后碰
   │  - 测试条件：碰撞速度48km/h，冲击方向与座椅纵向呈0°
   │  - 约束要求：座椅靠背抗后移量≤100mm（ECE R129 §7.1.4）
   │  - 合格判据：假人颈部剪切力≤500N，拉伸力≤1.5kN
   ├─ 动态碰撞：侧碰
   │  - 测试设备：移动侧撞壁障（质量770kg），碰撞速度60km/h
   │  - 侧防系统：必须包含能量吸收装置（如EPP泡棉+金属防撞梁）
   │  - 合格判据：假人胸部压缩≤44mm，头部横向位移≤150mm
   └─ 阻燃要求
      - 适用材料：座椅面料、头枕泡沫、安全带织带
      - 测试标准：ISO 3795:2019（汽车内饰材料阻燃性）
      - 合格判据：水平燃烧速度≤100mm/min，无熔融滴落引燃下方棉花
```

## 技术规范

### 输出格式要求
1. 标准信息必须包含：
   - 标准名称和版本
   - 实施要求
   - 核心要求摘要

2. 假人参数必须包含：
   - 假人模型名称
   - 百分位和年龄
   - 标准编号（如 ISO 13232-2:2021）
   - 身高体重范围
   - 人体测量参数（坐高、肩宽、头围）
   - 安装方向及依据

3. 设计参数必须包含：
   - 头枕高度（调节范围、基准点、公差）
   - 座宽（有效座宽、总座宽）
   - ISOFIX参数（尺寸等级、长度宽度、固定点间距、公差）
   - 侧防参数（面积、覆盖区域、测试标准）

4. 测试要求必须包含：
   - 正面碰撞（速度、加速度、HIC限值、标准条款）
   - 侧撞（速度、胸部压缩量、压缩速度、标准条款）
   - 安全带强度（纵向、横向、测试标准）

5. 标准测试项必须包含：
   - 正碰（设备、姿态、判据）
   - 后碰（条件、要求、判据）
   - 侧碰（设备、侧防系统、判据）
   - 阻燃（材料、标准、判据）
